# WhormholeEngine
Wormhole Engine indexed and show in indipendents wormholes a preview of web pages based on search frequency and the links between them

The Wormhole Engine displays the web pages that are typically searched for during a comprehensive search on a topic; for example, 
if people search for the word “car,” the Wormhole Engine displays the web pages most frequently searched for by users, 
rather than those that have paid to appear at the top of the results, Therefore, the Wormhole Engine database contains 
multiple independent wormholes. For example, a search for “car” might include a wormhole where the pages range from race 
cars to compact cars, with the pages arranged in a logical sequence based on what users typically search for—such as insurance, 
hail-proof covers, and so on.

Thus, the Wormhole Engine does not function like Google, which displays a list of indexed pages based on who pays the most; rather, pages are indexed based 
on search frequency and the links between them, which are assigned greater weight in the database and can therefore be represented as a graph.
